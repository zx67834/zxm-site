# 伪终端应该留下什么

真正的终端有权限、进程、网络和风险。

这个终端没有。它更像一张可以输入的个人名片：

- `whoami` 会介绍这个站
- `ls` 与 `cat` 可以展示少量公开信息
- `open blog` 会带你回到文章区
- `ai` 只在访客主动添加自己的 Key 后可用

## 为什么要保留它

因为个人网站不必只有卡片和链接。一个很小的交互入口，足够让访客感到这里确实有人在写、在做、在持续维护。

```bash
visitor@zxm-site:~$ whoami
zxm · builder / writer / learner
```

> 没有后端，也不读取访客的本地文件。
